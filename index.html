مقدمة عن البرمجة

ما هي البرمجة؟

البرمجة هي عملية إنشاء تعليمات (Instructions) للكمبيوتر لحل مشكلة معينة أو أداء مهمة محددة. فكر في البرمجة كطريقة للتحدث مع الكمبيوتر بلغة يفهمها.

مثال بسيط: عندما تريد من هاتفك تشغيل أغنية معينة، فأنت تعطيه تعليمات (اضغط على التطبيق، ابحث عن الأغنية، اضغط تشغيل). البرمجة تشبه هذا لكن بلغة الكمبيوتر.

لماذا نتعلم البرمجة؟

  • فرص عمل لا محدودة: الطلب على المبرمجين في ازدياد مستمر
  • راتب عالي: المبرمجون من أصحاب الرواتب العالية في السوق
  • تطوير الذكاء: البرمجة تحسن من قدرتك على حل المشكلات
  • الإبداع: يمكنك إنشاء أي شيء تتخيله (مواقع، تطبيقات، ألعاب)
  • المستقبل: كل شيء حولنا أصبح رقمياً ويحتاج للبرمجة

أنواع البرمجة

1

برمجة المواقع (Web Development)

إنشاء مواقع إنترنت مثل فيسبوك، جوجل، أمازون. تنقسم إلى:

  • Front-end: ما يراه المستخدم (التصميم والتفاعل)
  • Back-end: ما يحدث خلف الكواليس (قاعدة البيانات، المنطق)
2

برمجة التطبيقات (Mobile Development)

إنشاء تطبيقات للهواتف الذكية:

  • Android: باستخدام Java أو Kotlin
  • iOS: باستخدام Swift
  • Cross-platform: تطبيق واحد لكل المنصات
3

برمجة سطح المكتب (Desktop)

البرامج التي تعمل على الكمبيوتر مثل Microsoft Word، Adobe Photoshop، وبرامج الألعاب.

4

البرمجة المدمجة (Embedded)

برمجة الأجهزة الذكية مثل أجهزة التلفزيون، السيارات الذكية، والأجهزة المنزلية.

لماذا Java تحديداً؟

  • سهلة التعلم: صممت لتكون بسيطة ومفهومة للمبتدئين
  • مطلوبة في السوق: أكثر اللغات استخداماً في الشركات الكبرى
  • متعددة الاستخدامات: يمكن استخدامها لكل شيء (مواقع، تطبيقات، برامج)
  • مجتمع كبير: ملايين المطورين حول العالم يستخدمونها
  • مجانية: لا تحتاج لدفع أي أموال لتعلمها أو استخدامها
حقيقة مثيرة: Java تستخدم في أكثر من 3 مليارات جهاز حول العالم! من هاتفك الذكي إلى أنظمة البنوك العالمية.

تثبيت وتجهيز الأدوات

الخطوة الأولى: تثبيت Java JDK

JDK تعني Java Development Kit - وهي الأدوات الأساسية لكتابة وتشغيل برامج Java

1

تحميل JDK

اذهب إلى موقع Oracle الرسمي أو OpenJDK وحمل أحدث إصدار من JDK

نصيحة: اختر الإصدار LTS (Long Term Support) لضمان الاستقرار
2

تثبيت JDK

  • Windows: شغل ملف .exe واتبع التعليمات
  • Mac: شغل ملف .dmg
  • Linux: استخدم package manager
3

التحقق من التثبيت

افتح Terminal/Command Prompt واكتب:

java --version javac --version

إذا ظهرت أرقام الإصدارات، معناها التثبيت تم بنجاح!

الخطوة الثانية: تثبيت IDE

IDE تعني Integrated Development Environment - بيئة التطوير المتكاملة

1

اختيار IDE المناسب

  • IntelliJ IDEA: الأكثر احترافية (مجاني للطلاب)
  • Eclipse: مجاني بالكامل ومشهور
  • NetBeans: بسيط ومناسب للمبتدئين
  • VS Code: خفيف وسريع مع إضافات Java
2

تثبيت IntelliJ IDEA (المفضل)

حمل النسخة المجانية (Community Edition) من الموقع الرسمي وثبتها

للطلاب: يمكنكم الحصول على النسخة المدفوعة مجاناً بإيميل الجامعة

الخطوة الثالثة: أول برنامج "Hello World"

1

إنشاء مشروع جديد

في IntelliJ: File → New → Project → Java → Next → Finish

2

إنشاء Class جديد

اضغط يمين على src → New → Java Class → اكتب "HelloWorld"

3

كتابة الكود

public class HelloWorld { public static void main(String[] args) { System.out.println("مرحبا بك في عالم Java!"); System.out.println("أهلا وسهلا، أول برنامج لي 🎉"); // هذا تعليق - لن يتم تنفيذه System.out.println("أنا أتعلم البرمجة!"); } }
4

تشغيل البرنامج

اضغط على الزر الأخضر ▶️ أو اضغط Ctrl+Shift+F10

مبادئ التفكير البرمجي

ما هو التفكير البرمجي؟

التفكير البرمجي هو طريقة لحل المشكلات بتقسيمها إلى خطوات صغيرة ومنطقية يمكن للكمبيوتر فهمها وتنفيذها.

مثال من الحياة: عندما تريد الذهاب للجامعة، لا تقول "اذهب للجامعة" بل تفكر: استيقظ → اغتسل → تناول الفطار → البس → اخرج من البيت → اركب المواصلات → انزل في الجامعة

الخوارزميات (Algorithms)

الخوارزمية هي سلسلة من التعليمات المحددة والواضحة لحل مشكلة معينة.

1

خصائص الخوارزمية الجيدة:

  • واضحة: كل خطوة مفهومة
  • محددة: لا يوجد غموض في التعليمات
  • منتهية: تتوقف بعد عدد محدود من الخطوات
  • فعالة: تحل المشكلة بأقل وقت وجهد
2

مثال: خوارزمية العثور على أكبر رقم

// خوارزمية إيجاد أكبر رقم من ثلاثة أرقام 1. اقرأ الرقم الأول (a) 2. اقرأ الرقم الثاني (b) 3. اقرأ الرقم الثالث (c) 4. افترض أن a هو الأكبر 5. إذا كان b أكبر من a، فـ b هو الأكبر 6. إذا كان c أكبر من الأكبر الحالي، فـ c هو الأكبر 7. اطبع الرقم الأكبر

التطبيق العملي: عمل الشاي

الهدف من التطبيق

سنتعلم كيف نحول عملية يومية بسيطة (عمل الشاي) إلى برنامج Java. هذا يعلمنا أساسيات التفكير البرمجي.

خطوات عمل الشاي (الخوارزمية)

1

تحضير المكونات

ماء، شاي، سكر، كوب، ملعقة

2

تسخين الماء

ضع الماء على النار حتى يغلي

3

إضافة الشاي

أضف الشاي إلى الماء المغلي واتركه ينقع

4

إضافة السكر

أضف السكر حسب الرغبة وحرك

5

التقديم

اسكب الشاي في الكوب واستمتع!

public class TeaMaker { public static void main(String[] args) { System.out.println("🍵 برنامج صنع الشاي 🍵"); System.out.println("======================"); // الخطوة 1: تحضير المكونات System.out.println("1️⃣ تحضير المكونات:"); System.out.println(" ✅ ماء"); System.out.println(" ✅ شاي"); System.out.println(" ✅ سكر"); System.out.println(" ✅ كوب"); // الخطوة 2: تسخين الماء System.out.println("\n2️⃣ تسخين الماء..."); System.out.println(" 🔥 الماء على النار"); System.out.println(" ⏳ انتظار حتى الغليان"); // الخطوة 3: إضافة الشاي System.out.println("\n3️⃣ إضافة الشاي:"); System.out.println(" 🌿 تم إضافة الشاي"); System.out.println(" ⏰ ترك الشاي ينقع لمدة دقيقتين"); // الخطوة 4: إضافة السكر System.out.println("\n4️⃣ إضافة السكر:"); System.out.println(" 🍯 تم إضافة السكر حسب الرغبة"); System.out.println(" 🥄 تحريك الخليط"); // الخطوة 5: التقديم System.out.println("\n5️⃣ التقديم:"); System.out.println(" ☕ تم سكب الشاي في الكوب"); // النتيجة النهائية System.out.println("\n🎉 تم تحضير الشاي بنجاح!"); System.out.println("📝 ملاحظة: تذكر أن تطفئ النار!"); System.out.println("\n======================"); System.out.println("😋 استمتع بشايك اللذيذ!"); } }
